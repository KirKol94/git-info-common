Начало работы над проектом

Для начала работы с git необходимо проинициализировать проект командой git init

Теперь мы можем работать над проектом Добавлять папки, файлы и работать в них

В процессе работы

Индексация файлов

После того, как мы проинициализировали проект и внесли какие-то изменения, необходимо проиндексировать файлы командой git add «filename»удобно использовать команду git add .индексирует сразу все папки и файлы

Сохранение изменений (commit)

Чтобы сохранить изменения необходимо выполнить команду git commit -m "branchname | description"где branchname - название ветки, в которой были сделаны изменения, description - описание изменений

Основы работы с терминалом

CMD в Windows

CMD неудобная штука, поэтому лучше использовать Powershell. Подробнее об этом можно почитать где-нибудь в интернете.

Powershell в Windows

Powershell мощный инструмент. На нём можно писать сложные скрипты под Windows, но нас пока интересуют основы.

Для начала научимся перемещаться по директориям, смотреть их содержимое.
Для перемешения по каталогам используем команду cd Для просмотра содержимого каталога можно использовать команду ls А путь до текущего каталога мы можем посмотреть командой pwd

Примеры для cd
cd .. - переход в каталог выше cd Documents - переход в каталог Documents cd Documents/GitHub - переход в каталог GitHub cd Documents/GitHub/Git-Project - переход в каталог Git-Project
Примеры для ls
ls - просмотр содержимого текущего каталога ls -a - просмотр содержимого текущего каталога со всеми файлами и папками ls -l - просмотр содержимого текущего каталога со всеми файлами и папками и информацией о них ls -l -a - просмотр содержимого текущего каталога со всеми файлами и папками и информацией о них
Примеры для pwd
pwd - просмотр текущего каталога
